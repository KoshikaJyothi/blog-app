# Blog App

A full-stack Blog Application built using the MERN stack where users can create, read, update, and manage blog posts.

## Features

- User Authentication (Login & Signup)
- Create Blog Posts
- Edit and Delete Blogs
- View All Blogs
- Responsive User Interface
- Backend API Integration
- MongoDB Database Connectivity

## Tech Stack

### Frontend
- React.js
- React Router
- Axios
- CSS / Bootstrap

### Backend
- Node.js
- Express.js

### Database
- MongoDB
- Mongoose

## Installation

### Clone the repository

```bash
git clone https://github.com/KoshikaJyothi/blog-app.git
```

### Navigate to project folder

```bash
cd blog-app
```

---

## Backend Setup

```bash
cd backend
npm install
npm start
```
``

---

## Frontend Setup

```bash
cd frontend
npm install
npm run dev
```
